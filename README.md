# ComboTree

## Request

**CPU flag**:

- *flush*: clflush OR clflushopt OR clwb
- *streaming store*: sse2 OR avx2 OR avx512vl
- *bit instruction*: bmi1 AND bmi2

**endian**: little endian

**C++ standard**: C++11
